Intro

Javascript is a cross platform scripting language. It is a smalll and lightweight language.

Javascript contains a standard library of objects, such as Array, Date, and Math. Javascript can be extended for a variety of purposes by supplementing it with a variety of objects such as:

  • Client side JS extends the core language by supplying objects to control a browser and its DOM.
  • Server side JS etends the core language by supplying objects relevant to running JS on a server.


Need to know

  • Good working knowledge of HTML.


JS and CSS

Javascript aka JS is a programming language that gives life or movement to the web page, whereas CSS is a markup language that gives color and vibe to a page, something that HTML cannot.


Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.
